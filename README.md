# 💫 About Me:
I’m a Senior Unity Game Developer and founder at Aqua Games with 7+ years of experience in Unity, C#, mobile game development, puzzle games, hyper-casual games, and game publishing. I’ve published 100+ mobile games and worked with publishers and studios including Voodoo, Homa, TapNation, CrazyLabs, Supersonic, BoomBit, Joypac, and MondayOff.<br><br>My work covers the full game development pipeline: prototyping, MVP development, gameplay systems, level logic, UI, optimization, monetization, store-ready releases, and live-game iteration. Several shipped titles have reached millions of downloads, including games that ranked in Google Play Top Charts.<br><br>I’m currently pursuing my MS at ULB Brussels, majoring in Machine Learning, Artificial Intelligence, and Data Science. I’m especially interested in combining AI, analytics, and game development to build smarter, more engaging, and more scalable game experiences.
